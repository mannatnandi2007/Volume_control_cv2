A Python-based project that controls system volume using hand gestures captured via webcam. The distance between the thumb and index finger is tracked in real time to increase or decrease volume.

Tech: Python · OpenCV · MediaPipe · PyAutoGUI

How to Run?
pip install opencv-python mediapipe pyautogui
python main.py

 Gestures
Fingers apart → Volume Up
Fingers close → Volume Down

 Note
Press Esc to exit.
